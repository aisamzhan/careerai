## Skill Requirements

Employers in Kazakhstan expect a combination of technical, digital, and interpersonal skills.

### Profession: Software Developer
#### Required skills
- JavaScript
- React or Angular
- Node.js
- Databases (SQL / NoSQL)
- Git
- API development

### Profession: Data Analyst
#### Required skills
- SQL
- Python
- Excel
- Data visualization tools
- Statistics
- Business analysis

### Profession: Accountant
#### Required skills
- Accounting principles
- Financial reporting
- Excel
- Accounting software (1C)

### Profession: Mechanical Engineer
#### Required skills
- Mechanical design
- CAD software
- Engineering mathematics
- Equipment diagnostics

### Profession: Teacher
#### Required skills

### Profession: Logistics Manager
#### Required skills
- Supply chain management

## General Skill Expectations (Cross-cutting)

### Technical Skills (examples by domain)
- Engineering: CAD software, industrial systems, mechanical design.
- IT: programming languages, databases, software frameworks.
- Finance: accounting systems, financial modeling, reporting tools.
- Healthcare: medical procedures, diagnostics, patient care.

### Digital Skills (common tools)
- Microsoft Excel
- Microsoft PowerPoint
- CRM systems
- Data analysis tools
- Project management tools

Note: Basic digital literacy is considered essential.

### Soft Skills
- Communication: ability to clearly explain ideas and collaborate with others.
- Teamwork: working effectively in groups and organizations.
- Problem Solving: analyzing situations and finding solutions.
- Time Management: prioritizing tasks and meeting deadlines.
- Adaptability: learning new technologies and adapting to change.

### Language Requirements (Kazakhstan context)
- Kazakh: required in government and public sector jobs.
- Russian: widely used in business communication.
- English: important for international companies and technology sectors; English proficiency significantly increases career opportunities in IT, finance, and multinational organizations.
