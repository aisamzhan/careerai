# CareerAI Knowledge Base (Kazakhstan)

Эта папка хранит знания о рынке труда Казахстана, которые CareerAI использует как основу для:
- анализа резюме и навыков студента,
- сравнения с требованиями профессий,
- оценки "fit" (насколько профиль подходит под роль),
- рекомендаций по развитию и обучению.

## Зачем разбивать на 3 файла
Меньшие тематические файлы проще поддерживать и обновлять, а ИИ легче выбирать релевантные куски.

Файлы:
- `kz_market_1_industries_and_professions.md`: отрасли и топ‑профессии (контекст и карта рынка).
- `kz_market_2_skills_requirements.md`: требования по навыкам (что нужно уметь).
- `kz_market_3_salaries_and_career_paths.md`: зарплаты, карьерные треки, обучение/курсы.

## Правила наполнения (важно для "ответов по факту")
- Указывай даты и географию: KZ в целом или конкретные города/регионы (Алматы/Астана и т.д.).
- Числа (зарплаты, диапазоны) добавляй только если они прямо есть в тексте/источнике.
- Если есть источники (сайты вакансий, отчеты), добавляй их ссылками и датой снятия данных.
- Пиши структурно: заголовки, короткие пункты, без длинной "воды".

## Как ИИ должен использовать базу
- Использовать базу как контекст и пересказывать своими словами (не копировать большие куски).
- Если ответа нет в базе, честно сказать "нет данных в knowledge base" и спросить/предложить, что добавить.
